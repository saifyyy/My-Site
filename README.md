# Saif Ur Rehman - Portfolio

This is my personal portfolio website built using **React**, **TypeScript**, and **Vite**.  
It showcases my work, projects, skills, experience, and education in a clean and responsive layout.

## Tech Stack
- React
- TypeScript
- Vite
- CSS3
- IntersectionObserver (for scroll animations)

## How to Run

```bash
# Clone the repository
git clone https://github.com/saifyyy/your-portfolio.git

# Go inside the folder
cd your-portfolio

# Install dependencies
npm install

# Start the project
npm run dev
